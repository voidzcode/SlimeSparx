Don't wanna to type your full name and school into a sketchy discord bot? Don't want 800K SRP in 10 seconds just so your English teacher doesn't get too suspicious? Here is a chrome extension that is extremely useful for quickly solving mini quizzes by asking Gemini Ai the answers to the questions. It's pretty easy to use as you just add the extension, open the side panel, and then paste the text and questions into Gemini. You can get over 500SRP in less than 20 minutes (rough estimate). This also works on sparx assessments if you want to use it on those. If you have any problems/issues please report them to the "Issues" tab!
