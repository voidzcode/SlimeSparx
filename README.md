Don't wanna to type your full name and school into a sketchy discord bot? Don't want 800K SRP in 10 seconds just so your English teacher doesn't get too suspicious? Here is a chrome extension that is extremely useful for quickly solving mini quizzes by asking Gemini Ai the answers to the questions. It's pretty easy to use as you just add the extension, open the side panel, and then paste the text and questions into Gemini. You can get over 500SRP in less than 20 minutes (rough estimate). It has support for Sparx Reader, Sparx Maths and Sparx Science. This also works on sparx assessments if you want to use it on those. If you have any problems/issues please report them to the "Issues" tab!

Sparx Maths:
Teachers can usually see:

Whether you completed the homework
Completion percentage
The date and time you handed it in
How long you spent on the homework
How many attempts you made on questions
Which questions you got wrong
Which topics you're struggling with
XP earned and reward data
Homework difficulty settings and homework length
Class and year-group reports with your progress over time
Question-level analysis reports for assessments and tests

Many teachers also report seeing timestamps and answer histories, which can make unusually fast completions or odd answer patterns stand out, though this isn't the same as proving cheating.
